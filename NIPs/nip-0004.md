Current QR-code specification doesn't support mosaics.
'''
{
    "v": 2,
    "type": 2,
    "data": {
        "addr": "",
        "msg": "",
        "name": "",
        "amount": 0
    }
}
'''

I propose QR-code specification which supports mosaics.
'''
{
    "v": ?,
    "type": ?,
    "data": {
        "addr": "",
        "msg": "",
        "name": "",
        "mosaics": [
            {
                "name": "nem:xem",
                "amount": 0
            },
            {
                "name": "lc:usd",
                "amount": 0
            }
        ]
        
    }
}
'''
